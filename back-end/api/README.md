# WATTever - EV Charging Management System
Software Engineering Project | NTUA 2025-2026

Αυτό το αρχείο περιέχει τις οδηγίες εγκατάστασης και εκτέλεσης για το Back-end REST API.

---

1. Προαπαιτούμενα
Θα χρειαστείτε εγκατεστημένα στον υπολογιστή σας:
* Python 3.8 ή νεότερη έκδοση
* MariaDB Server (τρέχει τοπικά στη θύρα 3306)

---

2. Εγκατάσταση Βιβλιοθηκών
Ανοίξτε τερματικό στον κεντρικό φάκελο του project και τρέξτε την εντολή:

pip install -r requirements.txt

---

3. Ρύθμιση Βάσης Δεδομένων
Πριν τρέξετε το API, πρέπει να δημιουργήσετε τη δομή της βάσης δεδομένων.

Βήμα 3α: Συνδεθείτε στη MariaDB
Ανοίξτε το τερματικό της MariaDB (ή γραμμή εντολών):

mariadb -u root -p

Βήμα 3β: Εκτέλεση του SQL Script
Μέσα στο περιβάλλον της MariaDB, τρέξτε την εντολή για να δημιουργηθούν οι πίνακες:

source database/schema.sql;

(Προσοχή: Βεβαιωθείτε ότι η διαδρομή του αρχείου είναι σωστή).

Βήμα 3γ: Ρύθμιση Κωδικών
Ανοίξτε το αρχείο 'api/main.py' και βεβαιωθείτε ότι τα στοιχεία στο λεξικό 'DB_CONFIG' (user, password) ταιριάζουν με τους κωδικούς της δικής σας βάσης.

---

4. Εκτέλεση του Server
Για να ξεκινήσετε το API, μπείτε στον φάκελο 'api' και τρέξτε την εντολή:

cd api
python3 -m uvicorn main:app --host 0.0.0.0 --port 9876 --reload

Ο server θα είναι πλέον διαθέσιμος στη διεύθυνση: http://localhost:9876

---

5. Αρχικοποίηση Δεδομένων (Υποχρεωτικό για την πρώτη φορά)
Όταν τρέξετε το API για πρώτη φορά, η βάση δεν θα περιέχει σταθμούς φόρτισης. Ακολουθήστε αυτά τα βήματα για να τη γεμίσετε:

1. Ανοίξτε τον browser στη διεύθυνση: http://localhost:9876/docs
2. Βρείτε το endpoint: POST /api/admin/resetpoints
3. Πατήστε το κουμπί 'Try it out' και μετά 'Execute'.

Αυτό θα διαβάσει το αρχείο 'parts1234.json' και θα εισάγει τους σταθμούς στη βάση.

---

6. Documentation και Χρήση
Το API παρέχει περιβάλλον Swagger UI για δοκιμές στη διεύθυνση:
http://localhost:9876/docs

Βασικά Endpoints:
* GET /api/admin/healthcheck : Έλεγχος καλής λειτουργίας
* GET /api/points : Λίστα σημείων φόρτισης
* POST /api/reserve/{point_id} : Κράτηση θέσης
* POST /api/newsession : Καταγραφή νέας φόρτισης
